# util

## 목적

여러 컨트롤러와 서비스에서 재사용하는 보조 유틸리티를 둔다.

## 주요 파일

| 파일명 | 설명 |
| --- | --- |
| Resp.java | 공통 응답 래퍼다. |
| Script.java | 브라우저 스크립트 문자열 생성을 돕는다. |

## 하위 디렉토리

- 없음

## AI 작업 지침

- 공통 유틸은 특정 도메인에 강하게 결합되지 않도록 유지한다.
- 화면 흐름용 스크립트 유틸을 수정할 때는 실제 브라우저 동작을 같이 확인한다.
- `Resp`는 JSON 응답 래퍼 역할만 유지하고, `Script`는 MVC 스크립트 문자열 생성 역할만 유지한다.
- `Script`는 메시지와 경로를 JavaScript 문자열로 escape 한 뒤 스크립트를 생성한다.

## 테스트

- 관련 변경 후 호출하는 컨트롤러의 응답 포맷과 브라우저 동작을 확인한다.
- `RespScriptTest`로 공통 응답 body와 스크립트 문자열 기준을 확인한다.

## 의존성

- 내부: 각 MVC 컨트롤러
- 외부: `Spring MVC`

## 1. 스킬 문서 경로 규칙

신규 스킬 문서는 아래 경로 규칙을 따른다.

1. 경로: `.docs/skills/NN-스킬명/`
2. 파일명: `NN-스킬명-가이드.md`
3. 번호 `NN`은 두 자리 숫자(예: `13`, `14`)를 사용

예시:
- `.docs/skills/13-스킬명/13-스킬명-가이드.md`

---

## 3. 작성 규칙

1. 문서는 한국어 기준으로 작성한다.
2. 적용 대상과 입력 정보는 모호한 표현을 금지한다.
3. 완료 기준은 측정 가능한 문장으로 작성한다.
4. 스킬 목표와 범위를 분리해 기술한다.
5. 출력 결과는 재현 가능한 형식으로 작성한다.

---

## 4. 작성 체크리스트

1. 스킬 적용 대상이 명확한가?
2. 입력/출력이 누락 없이 정의되었는가?
3. 절차가 순서대로 실행 가능하게 작성되었는가?
4. 완료 기준이 측정 가능한가?
5. 관련 운영 문서와 충돌하지 않는가?
